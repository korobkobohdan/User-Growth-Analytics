# User-Growth-Analytics
# User Growth Analytics Dashboard

## Opis projektu

Projekt przedstawia analizę zachowań użytkowników aplikacji z wykorzystaniem narzędzia Amplitude.

Celem projektu było zbadanie procesu onboardingu, retencji użytkowników oraz identyfikacja etapów, na których użytkownicy najczęściej rezygnują z korzystania z aplikacji.
---
## Cele analizy
- Analiza procesu onboardingu
- Analiza lejka konwersji (Funnel)
- Analiza retencji użytkowników
- Porównanie zachowań użytkowników iOS oraz Android
- Identyfikacja miejsc powodujących spadek konwersji
---
## Wykorzystane narzędzia
- Amplitude
---
## Najważniejsze wnioski
- Zidentyfikowano około **50% spadek użytkowników na etapie rejestracji**.
- Porównano wskaźniki **Active Events** oraz **Content Playback** pomiędzy platformami iOS i Android.
- Wskazano etapy wymagające optymalizacji procesu onboardingu.
